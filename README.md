# サブ樹木医システム

樹木医の実務を補助する統合システム。NotebookLM「樹木医必携」の2冊
（**樹木医の手引き 第4版** ／ **緑化樹木腐朽病害ハンドブック**）を土台に、
外観診断から精密診断・危険度判定・処置までを支援する。

> **原則：AIは候補出し・整理・観察ガイドの助手。診断と処置の最終判断・責任は樹木医が負う。**

## 構成（3層）

1. **HTML診断支援ツール**（`public/index.html`）
   現地でオフライン参照できる単体ツール。
   - EBD標準診断フロー（5段階チェックリスト、進捗保存）
   - t/R比による危険度判定（断面ビジュアル・衰退度ランク→措置）
   - 精密診断機器リファレンス（レジストグラフ／PiCUS／γ線／生長錐＋フラクトメーター／エアースコップ）
   - 基礎知識ベース（生理生態とCODIT・構造力学・病害・虫害・環境ストレスと土壌・剪定・倫理）
   - 主要木材腐朽菌クイックリファレンス（同定は下記アプリへ連携）

2. **対話診断プロトコル**（`docs/dialog-protocol.md`）
   ベテラン樹木医（サブ樹木医）として、2冊を根拠に出典付きで診断支援する運用手順。

3. **土台ナレッジベース**（`docs/knowledge-base.md`）
   2冊から抽出した実務要約。NotebookLMのマインドマップ（`docs/mind-map.json`）等と連携。

## 腐朽菌の同定

腐朽菌の詳細同定は姉妹アプリ **[wood-decay-fungi](https://github.com/treedoctor360/wood-decay-fungi)**
（65種マスター＋写真AI候補出し＋記録同期）が担当。本システムはそれ以外の
診断ワークフローと基礎知識を担い、相互に連携する。
