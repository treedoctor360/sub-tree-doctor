# コーパス再生成時に取り込むOCR訂正（未反映リスト）

現行の `knowledgeEmbeddings.json` は**訂正前OCR**から生成されている。ここに挙げる訂正は
**次にコーパスを再生成する時**に、OCRマスター（`handbook_full.md` = 緑化樹木腐朽病害ハンドブック。
著作物のため非コミット）へ反映してから `npm run build-embeddings` を回すこと。

- 本ファイルは「何を直したか」の変更ログのみ（著作物本文は載せない）。
- 訂正の実体（正版の検索表・種一覧）はOCRマスター側に保持する。
- サブ樹木医は種の“確定”を wood-decay-fungi に委ねる設計のため、この訂正**単独での再生成は不要**
  （費用対効果が低い）。別件の再生成に相乗りで反映する。反映の本命は wood-decay-fungi アプリ側。
